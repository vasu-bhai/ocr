API Reference
Base URL: http://localhost:8000
POST /api/upload
Upload a document. Form data: file. Returns doc_id.
GET /api/status/{doc_id}
Poll processing status. Returns: pending / processing / done / review / failed
GET /api/result/{doc_id}
Get full extracted JSON with fields, confidence scores, and line items.
POST /api/correct/{doc_id}
Submit human corrections. Body: {"corrections": {"field": "value"}}
GET /api/documents
List all documents. Query params: status, skip, limit
GET /api/review-queue
Documents needing human review.
GET /api/stats
System statistics: totals, accuracy rate, queue depth.
DELETE /api/documents/{doc_id}
Delete a document and its extracted data.
Example: Upload via cURL
curl -X POST http://localhost:8000/api/upload \
  -F "file=@invoice.pdf"

# Response:
# {"doc_id": "abc-123", "status": "queued"}

curl http://localhost:8000/api/result/abc-123
